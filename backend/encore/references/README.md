## Key Concepts

### SDK Packages

**Go:**
- `encore.dev/api` - API endpoint definitions
- `encore.dev/storage/sqldb` - SQL database support
- `encore.dev/storage/cache` - Caching support
- `encore.dev/pubsub` - Pub/Sub messaging
- `encore.dev/config` - Secrets and configuration
- `encore.dev/cron` - Scheduled jobs
- `encore.dev/beta/auth` - Authentication
- `encore.dev/beta/errs` - Error handling
- `encore.dev/rlog` - Structured logging
- `encore.dev/et` - Testing utilities

**TypeScript:**
- `encore.dev/api` - API endpoint definitions
- `encore.dev/storage/sqldb` - SQL database support
- `encore.dev/storage/cache` - Caching support
- `encore.dev/pubsub` - Pub/Sub messaging
- `encore.dev/config` - Secrets and configuration
- `encore.dev/cron` - Scheduled jobs

## Deployment Targets

- **Encore Cloud**: Managed hosting with automatic scaling
- **Self-hosted**: Docker, Kubernetes, or custom infrastructure
- **AWS/GCP/Azure**: Build and deploy to major cloud providers

## Related Tools

- **Encore CLI**: Command-line interface for local development
- **Local Dashboard**: http://localhost:9400 for development insights
- **Client Generation**: Generate TypeScript/Go clients from API definitions
