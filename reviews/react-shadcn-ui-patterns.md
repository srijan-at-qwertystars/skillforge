Excellent shadcn/ui guide. Covers fundamentals (copy-paste model, Radix primitives, Tailwind CSS, CVA, cn()), components.json configuration, installation (npx shadcn@latest init/add, custom registry, monorepo), theming (CSS variables, dark mode, custom palette, next-themes), component anatomy (Radix + CVA + Tailwind + cn + forwardRef, asChild/Slot pattern), core components (Card, Dialog, DropdownMenu, Tabs), form patterns (react-hook-form + zod + shadcn Form components, cross-field validation, dynamic fields with useFieldArray), data table (TanStack Table integration with column defs, sorting, filtering, pagination, row selection), combobox/command (cmdk-based searchable lists), toast/Sonner notifications, sheet and dialog patterns (controlled dialog, responsive drawer/dialog with vaul, sheet side panels), and customization (custom CVA variants, LoadingButton wrapper pattern).
