Comprehensive testing strategy guide with standard description format. Covers test shapes (pyramid/Cohn, trophy/Dodds, diamond) with selection guidance, FIRST properties, AAA and Given-When-Then patterns, integration testing (Testcontainers, API tests), test doubles (dummy/stub/spy/mock/fake with decision rules, "never mock what you don't own"), contract testing (Pact consumer-driven contracts), property-based testing (patterns: round-trip/idempotency/invariant/oracle/metamorphic, fast-check/Hypothesis), mutation testing (Stryker/PIT, equivalent mutants, 70-85% target), snapshot testing with anti-patterns, TDD workflow (red-green-refactor, outside-in vs inside-out, when NOT to TDD), test organization (naming, file structure, factories), flaky test management (causes/detection/quarantine), coverage metrics (line/branch/condition/MC/DC, coverage ratchet), CI optimization (parallelization, Jest sharding, TIA, selective testing pipeline), and anti-patterns (testing implementation details, brittle tests, ice cream cone).
