# Review: tailwind-css-patterns

Accuracy: 5/5
Completeness: 5/5
Actionability: 5/5
Trigger quality: 5/5
Overall: 5.0/5

Issues: none

Outstanding Tailwind CSS guide with strong v4 coverage. Standard description format. Covers Tailwind v4 changes (CSS-first @theme configuration, Lightning CSS engine, @import "tailwindcss", @utility/@custom-variant directives, automatic content detection), core concepts (utility-first workflow, responsive mobile-first prefixes, state variants with group/peer), layout utilities (flexbox, grid with arbitrary values, spacing, positioning), typography (@tailwindcss/typography prose plugin), colors and theming (custom @theme colors, opacity modifiers, runtime theming), dark mode (class strategy, media strategy, semantic token pattern), responsive design (container queries, custom breakpoints, fluid typography with clamp), component patterns (extraction over @apply, CVA with class-variance-authority, when to use @apply for CMS content), animations (built-in + custom keyframes via @theme), custom configuration (extending theme, custom functional utilities with @utility, custom variants, legacy plugins), React/Vue integration (cn utility with clsx+tailwind-merge, conditional classes, className forwarding, Vue class binding), performance (auto content detection, no dynamic class construction), and anti-patterns.
