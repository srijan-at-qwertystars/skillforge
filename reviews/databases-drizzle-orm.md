Comprehensive Drizzle ORM guide. Covers schema definition (pgTable/mysqlTable/sqliteTable, identity columns, reusable timestamps pattern), relations (one-to-many/many-to-many/self-referencing, relations vs references distinction), SQL-like query builder (select/filter/join/aggregate/subquery), insert (single/bulk/upsert), relational queries (db.query with nested with), Drizzle Kit (generate/migrate/push/introspect/studio), transactions (nested savepoints, type extraction), prepared statements, database drivers (node-postgres/Neon/mysql2/better-sqlite3/Turso), type safety ($inferSelect/$inferInsert/customType), performance patterns, framework integration (Next.js/Hono/Express), testing, Drizzle vs Prisma comparison, and anti-patterns.
