# WXT Skill

**Status:** enriched

## Description

Web Extension framework for building browser extensions. Use for cross-browser extensions.
NOT for single-browser specific extensions without cross-browser needs.

## Contents

- **SKILL.md** - Main skill documentation with comprehensive WXT patterns
- **references/** - Official documentation links and GitHub repositories
  - `links.md` - Curated list of WXT docs, repos, and community resources
- **scripts/** - Helper scripts for WXT development
  - `init-project.sh` - Initialize a new WXT extension project
  - `dev-watch.sh` - Run development server with auto-reload
  - `build-all.sh` - Build for all target browsers
  - `package-extension.sh` - Package for distribution
  - `validate-config.sh` - Validate WXT configuration

- Chrome (Manifest V3)
- Firefox (Manifest V2/V3)
- Edge (Chromium)
- Safari
- Any Chromium-based browser
